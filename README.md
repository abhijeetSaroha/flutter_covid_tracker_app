# flutter_covid-19_tracker

A COVID tracker app built using the Flutter framework and a REST API is a mobile application that allows users to stay updated on the current number of COVID-19 cases in various countries. The app utilizes real-time data from reliable sources provided by the REST API to provide accurate information on the number of confirmed cases, deaths, and recoveries. The app also features a user-friendly interface, making it easy for users to navigate and find the information they need.  The REST API is used to fetch the data from the server and to display it on the app's interface.
